# 💰 NatSafeInvest - Assistente Virtual de Aplicações Financeiras

O **NatSafeInvest** é um assistente virtual inteligente desenvolvido como o Projeto Final do Bootcamp **Accenture - Python para Análise e Automação de Dados** na plataforma **DIO.me**. 

O objetivo principal do agente é democratizar o acesso à educação financeira, explicando conceitos de investimentos de forma clara, didática e segura, além de orientar simulações de aplicações financeiras com base em fontes de dados extremamente confiáveis.

---

## 📋 Estrutura do Projeto (Entregas do Desafio)

Este repositório foi construído seguindo rigorosamente as etapas de desenvolvimento de agentes de IA exigidas pelo desafio:

* **Etapa 1: Documentação do Agente:** Definição de escopo, objetivos e arquitetura do assistente (este arquivo `README.md`).
* **Etapa 2: Base de Conhecimento:** Fontes de dados e documentos oficiais utilizados para ancorar as respostas do modelo.
* **Etapa 3: Prompts do Agente:** Engenharia de prompt aplicada para definir a persona, tom de voz e restrições do assistente.
* **Etapa 4: Aplicação Funcional:** Implementação prática utilizando a plataforma **Google NotebookLM**.
* **Etapa 5: Avaliação e Métricas:** Testes de validação de respostas e cenários simulados.
* **Etapa 6: Pitch:** Resumo dos resultados e valor gerado pelo agente.

---

## 🎯 Escopo e Objetivos do NatSafeInvest

* **Desmistificar Investimentos:** Traduzir termos técnicos do mercado financeiro (como CDI, IPCA, Liquidez, Renda Fixa e Variável) para uma linguagem simples e acessível.
* **Ancoragem de Respostas (Grounding):** Eliminar totalmente alucinações de IA, respondendo dúvidas exclusivamente com base nos materiais oficiais carregados na base de conhecimento.
* **Orientação Segura:** Fornecer simulações conceituais sem realizar recomendações diretas de compra, priorizando a segurança financeira do usuário.

---

## 🛠️ Tecnologias Utilizadas

* **Google NotebookLM:** Plataforma utilizada para a criação da base de conhecimento e interface de interação com o agente.
* **Markdown:** Para estruturação profissional da documentação no GitHub.
